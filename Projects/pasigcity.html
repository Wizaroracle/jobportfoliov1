Pasig City Covid-19 Monitoring System


-Heavily utilized PHP and JS as the main programming language to develop the Pasig City COVID-19 Monitoring System, creating a web-based application connected to MySQL database


-Key features include a login panel for both users and admins, enabling admins to execute the CRUD system for managing barangay cases and adding new admin accounts. Users can access a view-only mode to monitor COVID-19 cases, displaying the percentage of affected patients from different barangays.

System Demo





A mock Up design for the Home Page



The admin section allows users with administrative privileges to perform CRUD operations on data, while the non-admin section only allows users to view data. Admins have full control over the data, while non-admin users can only consume it. This separation of roles ensures data integrity and security within the application.





The table displays the total number of COVID-19 cases for each barangay. (Mock up design)





The data provides information about the listed barangays, including various details such as confirmed cases, recovered cases, death, active, and date of case.





The admin is granted exclusive access to the barangay panel, enabling them to insert new data. This special privilege ensures that only authorized administrators can add information to the panel, maintaining data integrity and security.





The cases panel is exclusively accessible to the admin, granting them the ability to insert new data. This ensures that only authorized administrators have the privilege to add information to the panel, ensuring data integrity and security.





The barangay list can only be edited by an admin, ensuring that only authorized administrators have the ability to make changes. This restriction safeguards the integrity and accuracy of the barangay list, maintaining data consistency within the system.





When the admin logs in, a new tab labeled "Logout" becomes available, allowing the admin to easily log out from the system. This feature provides a convenient and secure way for the admin to end their session and maintain the privacy and access control of the admin account.
